# DX Scanner

#### Scan your project for possible DX recommendations.

DX Scanner allows you to ‚Äúmeasure‚Äù Developer Experience directly based on your source code and will recommend practices to adopt that will help you to improve your product development.

## Getting Started

### Installation

- with NPM
  
  ```npm install -g dxscanner``` 
- with Yarn 
  
  ```yarn global add dxscanner```

### Usage

**Scan repository**
```
dxscanner https://github.com/DXHeroes/dx-scanner
```
**Scan local path**
```
dxscanner ~/my-project
```

**All options**

```
dxscanner [path]
```
